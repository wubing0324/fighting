// 4.可以防止意外的类型错误
// 类型推断（不带类型）：
// 1.TypeScript具有强大的类型推断能力
// 2.当数据结构简单且明确时，TypeScript可以自动推断出正确的类型
// 3.代码更简洁
// 4.对于简单的数据结构来说足够用

// cutLists没有显式定义类型，因为TypeScript可以从初始值自动推断出它是一个包含{people: string, id: number, name: string}类型对象的数组
