1. Partial:将所有类型变为可选的
```javascript
interface User {
  id: number
  name: string
  age: number
  email: string
}
```
如果使用 Partial<User>，则生成的新类型将是所有属性都是可选的：
```javascript
type PartialUser = Partial<User>

```
PartialUser 的定义相当于：
```javascript
interface PartialUser {
  id?: number
  name?: string
  age?: number
  email?: string
}

```
