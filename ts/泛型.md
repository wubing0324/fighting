typescript提供的数据类型为：number、stirng、any、boolean、object等等，可以用这些数据类型来约束数据，而泛型也是一种约束条件，不过是开发者自定义的。


```javascript
const u = {
    name : 'wubing',
    age: 30,
    location: 'fdsfdsfds'
}

function fun<T>(obj: T, propName: keyof T) {}

fun(u, 'location')

```
