# CSDN算法题目下载指南

由于我无法直接访问CSDN网站获取内容，您需要手动下载这些文章。以下是一些建议的方法：

## 方法一：手动保存网页

1. 访问每个题目的CSDN链接，例如：https://blog.csdn.net/qfc_128220/article/details/127607871
2. 在浏览器中使用"另存为"功能，保存为HTML格式
3. 将保存的HTML文件放到 `od/原始文章/` 目录下
4. 运行 `od/csdn文章下载助手.js` 脚本处理下载的内容

## 方法二：使用浏览器扩展

1. 安装浏览器扩展如"SingleFile"或"Save Page WE"
2. 使用扩展下载完整网页
3. 将保存的文件放到 `od/原始文章/` 目录下
4. 运行处理脚本

## 方法三：使用命令行工具

如果您熟悉命令行工具，可以使用如下工具：

```bash
# 安装wget (Mac用户可以通过brew安装)
brew install wget

# 下载文章
wget -O od/原始文章/127607871.html https://blog.csdn.net/qfc_128220/article/details/127607871
```

## 提取文章ID

为了批量处理，请确保每个题目都有正确的文章ID。您可以从链接中提取ID，例如：
- 链接：`https://blog.csdn.net/qfc_128220/article/details/127607871`
- ID：`127607871`

请更新 `od/csdn文章下载助手.js` 文件中的文章ID，然后再运行脚本处理。
